# Website

[Bzlparty website](https://bzlparty.pages.dev/)

**NPM**

```bash
npm install
npm run serve
```

**pnpm**

```bash
pnpm install
pnpm serve
```

**Yarn**

```bash
yarn install
yarn serve
```

Run from Bazel:

```bash
bazel run //:serve
```

## Development

Install git hooks:

```bash
pre-commit install
```
